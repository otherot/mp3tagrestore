# MP3 Tag Restore

Скрипт для массового исправления кодировки ID3-тэгов в MP3-файлах.

## Проблема

Многие MP3-файлы имеют некорректную кодировку тэгов: текст сохранён в кодировке **Windows-1251** (кириллица), но помечен как **LATIN1**. Это приводит к отображению «кракозябр» вместо нормального текста:

- `Âàðèàíò Ê` вместо `Вариант К`
- `ß òåáÿ ëþáëþ` вместо `Я тебя люблю`
- `Ïîëåòàåì` вместо `Полетаем`

## Решение

Скрипт читает байты тэгов как Windows-1251 и пересохраняет их в **UTF-8**, который корректно отображается во всех современных плеерах.

## Требования

- Python 3.6+
- Библиотека `mutagen`

## Установка

```bash
pip install mutagen
```

## Использование

### Исправить все MP3 в текущей папке (рекурсивно)

```bash
python fix_mp3_tags.py . -r
```

### Исправить конкретную папку

```bash
python fix_mp3_tags.py "путь\к\папке" -r
```

### Исправить один файл

```bash
python fix_mp3_tags.py "трек.mp3"
```

### Сначала посмотреть, что будет исправлено (без записи)

```bash
python fix_mp3_tags.py "путь\к\папке" -r --dry-run
```

## Параметры командной строки

| Параметр | Описание |
|----------|----------|
| `path` | Путь к файлу или директории (по умолчанию: текущая папка) |
| `-r`, `--recursive` | Рекурсивно обрабатывать поддиректории |
| `--dry-run` | Только показать изменения, не записывать |

## Примеры вывода

```
============================================================
Файл: Вариант К - Я тебя люблю.mp3
============================================================
  TALB: 'Ïîëåòàåì' → Полетаем
  TIT2: 'ß òåáÿ ëþáëþ' → Я тебя люблю
  TPE1: 'Âàðèàíò Ê' → Вариант К
  TPE2: кодировка Encoding.UTF16 — OK
  ✓ Тэги исправлены и сохранены
```

## Исправляемые поля

Скрипт обрабатывает следующие ID3-тэги:

- **TALB** — Альбом
- **TIT2** — Название трека
- **TPE1** — Артист
- **TPE2** — Альбом артист
- **TCON** — Жанр
- **TDRC** — Год
- **TRCK** — Номер трека
- **TPOS** — Номер диска
- И другие текстовые поля
